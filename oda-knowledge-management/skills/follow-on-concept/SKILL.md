# 후속사업 개념서 작성

현 사업의 성과를 기반으로 후속사업 또는 확장사업의 개념서(Concept Note)를 작성한다.

## 왜 후속사업 기획이 중요한가

ODA 사업은 단발성으로 끝나면 효과가 약화된다. 1기 사업에서 구축한 기반을 2기에서 심화하고, 다른 국가로 확산하는 것이 UBION의 사업 파이프라인을 확보하는 핵심 전략이다. 발주처도 기존 사업의 성과가 입증된 후속사업을 선호한다.

## 후속사업 유형

### 1. 심화형 (Deepening)
- 동일 국가, 동일 분야에서 범위/깊이 확대
- 예: 1기 교원 연수 200명 → 2기 전국 확대 2,000명 + 인증 체계

### 2. 확장형 (Scaling)
- 동일 모델을 다른 지역/국가에 적용
- 예: 카자흐스탄 KSP → 우즈베키스탄 적용

### 3. 발전형 (Evolving)
- 1기 성과를 기반으로 새로운 영역 진출
- 예: 1기 LMS 구축 → 2기 AI 기반 적응형 학습 도입

### 4. 연계형 (Linking)
- 다른 발주처/재원으로 유사 사업 기획
- 예: KOICA 1기 → ADB TA로 확대

## 개념서 표준 구조

```
# 후속사업 개념서: [사업명(안)]

## 1. 배경
### 1.1 현 사업 성과 요약
### 1.2 후속사업 필요성

## 2. 사업 개요
- 사업명/목적/수원국/기간/예산/발주처 (안)

## 3. 핵심 전략
- 1기 대비 차별화/발전 포인트
- 접근 방법론/핵심 활동/기대 성과

## 4. 실현 가능성
- UBION 역량/수원국 환경/재원/리스크

## 5. 추진 로드맵
```

## 재원 확보 전략

- **동일 발주처 연속**: KOICA 1기 → KOICA 2기
- **발주처 다변화**: KOICA → ADB, UNESCO, World Bank
- **재원 혼합**: ODA + 수원국 정부 + 민간
- **PPP 모델**: UBION 자체 + KOICA 매칭
- **지역 확장**: 단일국 → 복수국 프로그램

## 작성 팁

- 1기 성과를 과장 없이 객관적으로 제시한다
- "왜 후속이 필요한가"의 논리가 명확해야 한다
- 발주처 담당자와 사전 비공식 협의를 거치면 성공률이 높아진다
- 수원국 정부의 공식 요청이 있으면 매우 유리하다

## 다음 단계 제안

- **확산 계획을 수립하려면** → dissemination-plan 스킬
- **역량이전 계획을 점검하려면** → capacity-transfer-plan 스킬
- **발주처 전략을 세우려면** → donor-strategy 스킬 (oda-partnership 플러그인)
