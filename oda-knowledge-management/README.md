# oda-knowledge-management

> ODA 지식관리 & 확산 플러그인 — 교훈, 성공사례, 사례연구, 확산 계획, 후속사업, 역량이전

## 스킬 (6)

| 스킬 | 설명 |
|---|---|
| `lessons-learned` | 교훈 추출 — 성공/개선 요인 분류, 일반화, 적용 제언 |
| `success-story` | 성공사례 — 수혜자 스토리텔링, 변화 전후, 홍보 자료 |
| `case-study` | 사례연구 — 학술적 구조, 맥락-개입-결과-분석-시사점 |
| `dissemination-plan` | 성과확산 — 대상별 채널, 메시지, 자료, 일정 |
| `follow-on-concept` | 후속사업 개념서 — 심화/확장/발전/연계, 재원 전략 |
| `capacity-transfer-plan` | 역량이전 — 4단계 이전, 자립 기준, 출구전략, 사후 모니터링 |

## 커맨드 (3)

| 커맨드 | 체이닝 | 설명 |
|---|---|---|
| `/extract-lessons` | lessons-learned → capacity-transfer-plan | 교훈 + 역량이전 점검 |
| `/write-case-study` | case-study → success-story | 사례연구 + 성공사례 |
| `/plan-follow-on` | follow-on-concept → dissemination-plan | 후속사업 + 확산 계획 |

## 설치

```bash
claude plugin install oda-knowledge-management@oda-skills
```
