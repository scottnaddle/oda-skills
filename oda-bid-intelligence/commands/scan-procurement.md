# /scan-procurement

조달공고를 분석하고, ODA 시장 트렌드와 연계하여 기회의 전략적 가치를 평가하는 워크플로우.

## 실행 흐름

### Step 1: 공고 분석 (procurement-monitor)

사용자가 제공한 공고 정보를 분석한다.

- 공고 텍스트가 첨부된 경우: 핵심 정보를 추출하여 요약 카드 생성
- 검색 조건이 제공된 경우: 조건에 맞는 공고를 찾기 위한 가이드 제공
- 복수 공고인 경우: 각각에 대해 요약 카드 생성 후 우선순위 매트릭스 작성

**산출물:**
- 공고별 요약 카드 (신호등 적합성 평가 포함)
- 복수 공고 시 우선순위 매트릭스

### Step 2: 시장 맥락 분석 (oda-landscape)

공고를 ODA 시장의 큰 그림 속에서 해석한다.

- 해당 분야/국가의 ODA 트렌드와 연결
- 후속 사업 가능성 평가
- UBION 포지셔닝 기회 식별

**산출물:**
- 해당 공고의 전략적 의미
- 후속 기회 전망
- 권고: 입찰 참여 여부 및 준비 액션

## 완료 후 제안

- 유망 공고 발견 시: "RFP를 상세 분석하시겠습니까? → `/bid-analyze`"
- 수원국 정보가 필요할 때: "해당 국가를 연구하시겠습니까? → `/research-country`"
