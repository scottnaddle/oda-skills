# ADB RFP 분석 가이드

아시아개발은행(ADB) 컨설팅 서비스 조달의 RFP를 분석할 때 참조하는 상세 가이드.

## 목차
1. ADB 조달 방식
2. 2단계 선발 프로세스
3. Technical Proposal 평가기준
4. Financial Proposal 구조
5. 핵심 유의사항

---

## 1. ADB 조달 방식

### 컨설팅 서비스 선발 방법
- **QCBS** (Quality and Cost Based Selection) — 가장 일반적. 기술 80: 가격 20 또는 기술 90: 가격 10
- **QBS** (Quality Based Selection) — 복잡한 사업. 기술만으로 평가 후 1위와 가격 협상
- **FBS** (Fixed Budget Selection) — 예산 고정. 예산 내 최고 기술점수 업체 선정
- **LCS** (Least Cost Selection) — 기술 통과 후 최저가. 단순/정형화된 업무에 사용
- **CQS** (Consultants' Qualifications Selection) — 소규모 사업. 자격 기반 직접 선정

### 적용 기준 (2024 Procurement Policy 기준)
- 국제경쟁입찰(ICB): $1M 이상 컨설팅
- 국내우선입찰(NCB): 해당국 내 조달
- 쇼핑: 소규모 ($100K 미만)

## 2. 2단계 선발 프로세스

### 1단계: EOI (Expression of Interest) & 숏리스트
- ADB의 Consulting Services Recruitment Notice (CSRN) 공고
- 컨설팅 업체가 EOI 제출 (기업 프로파일, 유사 실적, 핵심 인력)
- ADB가 6개사 내외로 숏리스트 작성
- 숏리스트에 포함되어야 RFP를 받을 수 있음

### 2단계: RFP → 제안서 제출 → 평가 → 계약
- 숏리스트 업체에만 RFP(Request for Proposals) 발송
- Technical Proposal과 Financial Proposal을 별도 봉투로 제출
- Technical Proposal을 먼저 개봉/평가
- 기술점수 통과자(보통 750점/1000점 이상)의 Financial Proposal만 개봉

### 타임라인 (통상)
- EOI 마감: 공고 후 3~4주
- 숏리스트 통보: EOI 후 4~8주
- RFP 발송: 숏리스트 후 2~4주
- 제안서 제출: RFP 후 6~8주
- 기술평가: 제출 후 4~8주
- 계약 협상: 기술평가 후 2~4주

## 3. Technical Proposal 평가기준

### 표준 배점 (QCBS, 1000점 만점 기준)

| 항목 | 배점 범위 | 핵심 |
|---|---|---|
| Specific Experience of the Firm | 50~100점 | 유사 분야/지역 프로젝트 실적 |
| Methodology and Work Plan | 200~400점 | 접근 방법론, 혁신성, 실현가능성 |
| Key Experts | 300~600점 | 개별 전문가의 자격, 경력, 적합성 |
| Knowledge Transfer | 50~100점 | 역량이전 계획, 현지화 전략 |

### Key Experts 평가 세부 (가장 높은 배점)
- **General Qualifications**: 학력, 전체 경력 연수
- **Specific Experience**: 유사 업무 경험, 해당 국가/지역 경험
- **Regional/Country Experience**: 수원국 또는 인근 국가 업무 경험

주의: ADB는 CV에 기재된 내용만 평가한다. 프로젝트명, 역할, 기간이 명확하지 않으면 경력으로 인정하지 않음.

### Methodology 평가 포인트
- **Technical Approach**: 문제 이해도, 접근 방법의 적절성
- **Work Plan**: 일정의 현실성, 마일스톤 설정
- **Organization and Staffing**: 팀 구성의 적절성, 현지 인력 활용
- **Innovation**: 기존 방법 대비 혁신적 접근 (점점 중시되는 추세)

## 4. Financial Proposal 구조

### 표준 비목 (ADB 양식)
- **Remuneration**: 전문가 보수 (International/National 구분)
  - International: $800~2,500/day (분야, 경력에 따라)
  - National: 현지 시장 기준
- **Reimbursable Expenses**:
  - Per Diem (숙박/식비)
  - International Travel
  - Local Transportation
  - Communications
  - Report Production
  - Equipment
- **Miscellaneous**: 기타 (visa, medical 등)
- **Contingency**: 통상 5~10% (별도 또는 포함)

### 가격평가 공식 (QCBS)
- Financial Score = 1000 × (Fm / F)
  - Fm = 최저 제안가격
  - F = 해당 업체 제안가격
- Combined Score = Technical Score × T% + Financial Score × F%
  - 통상 T=80, F=20 또는 T=90, F=10

## 5. 핵심 유의사항

### ADB 특유의 요구사항
- **Conflict of Interest**: ADB 프로젝트의 COI 규정이 매우 엄격. 동일 프로젝트의 준비 단계에 참여한 업체는 실행 단계 입찰 불가
- **Eligibility**: ADB 회원국 국적 업체만 참여 가능. 한국은 ADB 회원국
- **JV(Joint Venture)**: JV 구성 시 각 파트너의 역할, 지분, 책임을 명확히 정의해야 함
- **Association**: JV보다 느슨한 형태. Lead Firm + Associate 구조. 계약은 Lead Firm과 체결

### 제안서 작성 팁
- ADB RFP는 페이지 수 제한이 있는 경우가 많음 (Technical Proposal: 50~80쪽)
- 양식(Forms)이 지정되어 있으면 반드시 해당 양식을 사용
- CV는 ADB 표준 양식을 사용 — 다른 양식 사용 시 감점 가능
- Methodology에서 ADB의 기존 유사 프로젝트를 참조하면 긍정적 평가

### 자주 실수하는 부분
- Financial Proposal에서 세금(Tax) 처리: ADB는 보통 세금 면제이므로 세금 포함 여부를 RFP에서 정확히 확인
- Key Expert의 "Availability": 제안서 제출 시 전문가의 가용성 확인서(Availability Statement) 필요
- EOI 단계에서 제출한 Key Expert와 RFP 단계의 Key Expert가 다른 경우 — 변경 사유 설명 필요
- Currency: 보통 USD, 현지 통화 병기 필요 시 환율 기준일 확인
