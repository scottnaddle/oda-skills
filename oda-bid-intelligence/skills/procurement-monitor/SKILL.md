# 조달공고 모니터링

ODA 및 ICT 분야 조달공고를 체계적으로 분석하여, 입찰 기회를 신속하게 식별하고 우선순위를 매긴다.

## 이 스킬의 역할

조달공고가 올라오면 짧은 시간 안에 Go/No-Go를 판단해야 한다. 공고 전문을 처음부터 끝까지 읽는 대신, 핵심 정보를 구조화하여 의사결정 속도를 높인다.

## 분석 절차

### 1단계: 공고 핵심 정보 추출

공고 텍스트 또는 URL에서 다음 정보를 추출한다:

| 항목 | 내용 |
|---|---|
| 사업명 | 국문/영문 |
| 발주기관 | KOICA, NIPA, NIA, 기타 |
| 공고번호 | 식별번호 |
| 공고일 / 마감일 | 남은 일수 표시 |
| 예산규모 | 총액, 연도별 (가능 시) |
| 사업기간 | 시작~종료 |
| 수원국/대상지역 | 해당 시 |
| 사업 분야 | 교육, ICT, TVET, 보건 등 |
| 조달 방식 | 경쟁입찰, 제한경쟁, 수의계약 등 |
| 참가자격 | 핵심 자격요건 요약 |
| 유사실적 요건 | 건수, 금액, 분야 |
| 컨소시엄 | 허용/필수/불가 |

### 2단계: UBION 적합성 빠른 판단

5가지 기준으로 신호등 평가:

| 기준 | 🟢 Go | 🟡 검토필요 | 🔴 No-Go |
|---|---|---|---|
| 분야 적합성 | 교육/ICT 핵심 분야 | 교육+타분야 융합 | 비관련 분야 |
| 자격 충족 | 모든 요건 충족 | 일부 불확실 | 핵심 요건 미달 |
| 예산 규모 | 적정 (5~50억) | 과소/과대 | 수익성 없음 |
| 일정 | 충분한 준비 기간 | 촉박하나 가능 | 물리적 불가 |
| 경쟁 강도 | 낮음 (니치) | 중간 | 극심 (레드오션) |

### 3단계: 기회 요약 카드

각 공고를 한 장의 요약 카드로 정리한다:

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📋 [사업명]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
발주처: [기관명]        예산: [금액]
마감일: [날짜] (D-XX)   기간: [XX개월]
수원국: [국가명]        분야: [분야]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
적합성: 🟢🟢🟡🟢🟡  종합: Go / 검토 / No-Go
핵심 포인트: [1줄 요약]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### 4단계: 복수 공고 우선순위 매트릭스

여러 공고를 동시에 검토할 때, 우선순위를 매긴다:

- **영향도** (예산 규모 × 전략적 중요도): High / Medium / Low
- **수주 가능성** (적합성 + 경쟁강도): High / Medium / Low
- **영향도 × 수주가능성** 매트릭스에서 우상단(HH) 공고를 최우선 검토

## 출력 형식

단일 공고 분석:
```
# 조달공고 분석: [사업명]
[핵심 정보 표]
[적합성 판단]
[권고사항]
```

복수 공고 종합:
```
# 조달공고 모니터링 요약 — [기간]

## 우선순위 매트릭스
[매트릭스 표]

## 공고별 요약 카드
[카드 1]
[카드 2]
...

## 권고: 이번 주기 액션 아이템
1. [최우선 공고] — 즉시 RFP 분석 착수
2. [검토 공고] — 추가 정보 확인 후 판단
3. [관망 공고] — 모니터링 지속
```
