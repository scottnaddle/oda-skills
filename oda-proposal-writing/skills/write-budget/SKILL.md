# 예산서 작성

ODA 사업의 예산서를 발주처 비목 체계에 맞게 체계적으로 작성한다.

## 왜 예산서가 중요한가

예산서는 제안서의 실행 가능성을 입증하는 핵심 문서다. 워크플랜의 활동 하나하나에 적정 비용이 배정되어야 하고, 비목 간 배분이 합리적이어야 한다. 가격평가에서 너무 높으면 점수가 떨어지고, 너무 낮으면 사업 수행이 불가능해진다.

## 작성 절차

### 1단계: 비목 체계 확인

발주처별 비목 체계가 다르므로 먼저 확인한다:
- KOICA: `references/koica-budget-rules.md`
- ADB: `references/adb-cost-categories.md`

### 2단계: 활동 기반 예산 산정 (Bottom-up)

워크플랜의 각 활동에 필요한 비용을 산정한다:
- 투입 인력 × 단가 × 기간 = 인건비
- 출장 횟수 × 항공료 + 숙박비 + 일비 = 여비
- 기자재 목록 × 단가 = 기자재비
- 워크숍 횟수 × 참석자수 × 단가 = 행사비
- 각종 현지 운영비

### 3단계: 비목별 집계 및 조정

Bottom-up 산정 결과를 비목별로 집계하고 총액을 확인한다:
- 인건비 비율 확인 (보통 전체의 40~60%)
- 여비 비율 확인 (보통 15~25%)
- 간접비/일반관리비 적용 (발주처 규정에 따라)
- 예비비 설정 (보통 5~10%)
- RFP 상한 예산 대비 확인

### 4단계: 가격 경쟁력 검토

- 경쟁 입찰인 경우, 가격평가 점수 영향 검토
- 단가의 합리성 (시장 단가 대비)
- 비목별 비율의 적정성
- 컨소시엄의 경우 파트너 간 비용 배분

## 출력 형식

```
# 예산서: [사업명]

## 1. 예산 총괄표
[비목별 합계 표]

## 2. 비목별 산출 근거
### 2.1 인건비
[전문가별 등급, 단가, M/M, 소계]
### 2.2 여비
[출장별 항공, 숙박, 일비 산출]
### 2.3 기자재/장비
[품목별 사양, 수량, 단가]
### 2.4 현지활동비
[워크숍, 인쇄, 통신 등]
### 2.5 일반관리비
[산출 근거]
### 2.6 예비비
[산출 근거]

## 3. 연도별 집행 계획
[연도별 비목별 표]

## 4. 가격 경쟁력 분석
[총액 적정성, 비목 비율, 경쟁력 평가]
```
