# ADB Technical Proposal 양식 가이드

## ADB Technical Proposal 표준 구조

ADB QCBS 방식의 Technical Proposal은 크게 3파트로 구성된다.

### Part A: Technical Approach and Methodology

**A1. Understanding of the Assignment**
- TOR에 대한 이해도 표현
- 주요 이슈와 과제 식별
- 접근 방향 설정 근거

**A2. Technical Approach**
- 전체 방법론 프레임워크
- 단계별 접근 전략
- 핵심 방법론/도구 상세
- 데이터 수집 및 분석 방법 (해당 시)
- 혁신적 접근 요소

**A3. Methodology**
- 구체적 실행 방법
- 기법(Techniques) 및 도구(Tools) 상세
- 품질 보증 방법
- 이해관계자 참여 방법

**A4. Work Plan**
- 전체 일정(Bar Chart / Gantt Chart)
- 마일스톤 및 성과물 제출 일정
- 주요 회의/워크숍/미션 일정
- 성과물(Deliverables) 목록 및 제출 시기

### Part B: Organization and Staffing

**B1. Organization**
- 팀 조직도
- 의사결정 구조
- 본사-현지 간 소통 체계
- 카운터파트와의 협업 구조

**B2. Team Composition**
- Key Experts와 Non-Key Experts 구분
- 각 전문가의 역할 및 투입 기간
- 전문가 간 업무 연계

**B3. Staffing Schedule**
- 월별 투입 일정표 (Bar Chart)
- Key Expert별 현지/재택 근무 일정
- 총 투입 Person-Months

### Part C: Comments/Suggestions (선택)

- TOR에 대한 개선 제안
- 추가 업무 범위 제안
- 방법론적 대안 제시

주의: TOR에 대한 비판이 아니라 건설적 제안으로 표현해야 한다.

## ADB 양식 (Forms)

ADB RFP에는 보통 다음 양식이 지정된다:

- **Form TECH-1**: Technical Proposal Submission Form (커버 레터)
- **Form TECH-2**: Organization and Experience (회사 프로파일)
- **Form TECH-3**: Comments/Suggestions on TOR
- **Form TECH-4**: Description of Approach, Methodology, and Work Plan
- **Form TECH-5**: Team Composition, Assignment, and Key Experts' Inputs
- **Form TECH-6**: Curriculum Vitae (CV) for Key Experts

반드시 ADB 지정 양식을 사용해야 한다. 자체 양식 사용 시 감점 가능.

## CV 작성 규칙

ADB CV는 TECH-6 양식을 따른다:

### 필수 포함 항목
1. Position Title and No. (TOR에서 지정된 포지션명)
2. Name of Expert
3. Date of Birth / Nationality
4. Education (학위, 전공, 대학, 졸업연도)
5. Membership in Professional Associations
6. Countries of Work Experience
7. Languages (reading, writing, speaking 수준)
8. Employment Record (최근순, 상세 기술)
9. Detailed Tasks Assigned (이 프로젝트에서의 구체적 역할)
10. Work Undertaken that Best Illustrates Capability (유사 경험 5~10건)

### CV 작성 팁
- **최대 3~5쪽** (간결하게)
- 모든 프로젝트에 **정확한 기간**(년/월) 기재
- **역할(Position)**과 **수행 업무(Tasks)**를 명확히 구분
- 해당 사업과 직접 관련된 경험을 상단에 배치
- 모호한 표현 지양: "participated in" 대신 "led the design of..."

## 분량 가이드

| 항목 | 권장 분량 |
|---|---|
| Technical Approach & Methodology | 20~35쪽 |
| Organization & Staffing | 5~10쪽 |
| Work Plan (Gantt Chart 포함) | 3~5쪽 |
| CV per Key Expert | 3~5쪽 |
| 전체 Technical Proposal | 50~80쪽 (CV 별도) |

## ADB 평가위원 특성

- ADB 평가위원은 보통 3~5인 (ADB 직원 + 외부 전문가)
- **Methodology 섹션에 가장 많은 배점** → 가장 공을 들여야 하는 부분
- Key Experts의 CV를 개별 채점 → CV 품질이 매우 중요
- Innovation과 Knowledge Transfer를 점점 더 중시하는 추세
- 명확하고 논리적인 서술 선호, 수사적 표현 지양
