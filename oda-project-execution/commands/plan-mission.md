# /plan-mission

현지 출장의 계획 또는 결과 보고를 지원하는 워크플로우.

## 실행 흐름

### 출장 전 (계획 모드)
1. 출장 목적 및 기대 성과 정의
2. 일정표 수립 (면담, 현장 방문, 워크숍)
3. 브리핑 자료 준비 (expert-dispatch 스킬 연계)
4. 로지스틱스 체크리스트 (travel-logistics 스킬 연계, oda-toolkit)

### 출장 후 (보고 모드)
mission-report 스킬을 실행한다.
1. 일정별 활동 요약
2. 면담 요약 (대상, 내용, 합의사항)
3. 현장 관찰 사항
4. 주요 발견 및 시사점
5. 후속조치 (담당자/기한)

## 완료 후 제안

- 진도보고에 반영하려면: → `/write-progress`
- 리스크를 업데이트하려면: → `/manage-risks`
