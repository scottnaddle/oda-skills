# /write-inception

사업 착수 단계에서 필요한 문서 패키지를 한 번에 작성하는 워크플로우.

## 실행 흐름

### Step 1: 착수보고서 (inception-report)

착수미션 결과를 반영한 착수보고서를 작성한다.
- 제안서 PDM 대비 현장 확인 결과 반영
- 수정 워크플랜 수립
- 베이스라인 조사 계획
- M&E 체계 확정

### Step 2: 리스크 등록부 (risk-register)

착수미션에서 확인된 리스크를 등록부로 작성한다.
- 현장에서 새로 식별된 리스크 추가
- 제안서 단계의 리스크 재평가
- 대응 전략 및 담당자 지정

### Step 3: 전문가 파견 계획 (expert-dispatch)

1차 파견 이후의 전문가 파견 일정을 확정한다.
- 워크플랜 기반 파견 일정
- 전문가별 TOR 재확인
- 카운터파트와의 일정 조율 포인트

## 완료 후 제안

- PSC 1차 회의를 준비하려면: → `/prepare-psc` (oda-partnership 플러그인)
- 베이스라인 조사를 설계하려면: → `/assess-needs` (oda-country-research 플러그인)
