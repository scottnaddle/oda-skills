# ADB Progress Report 양식

## Quarterly Progress Report (QPR) 구조

```
Quarterly Progress Report
Project: [사업명 및 번호]
Reporting Period: [기간]

1. Executive Summary
   - Overall project status (On Track / At Risk / Off Track)
   - Key achievements this quarter
   - Key issues requiring attention

2. Progress Toward Outcomes
   - DMF indicator tracking table
   | Indicator | Baseline | Target | Current | Status |
   - Analysis of progress toward each outcome

3. Progress on Outputs
   3.1 Output 1: [title]
       - Planned activities vs completed
       - Key deliverables produced
       - Issues and corrective actions
   3.2 Output 2: [title]
       ...

4. Implementation Status
   4.1 Consulting Services
       - Expert mobilization status
       - Person-months utilized vs planned
   4.2 Procurement
       - Equipment/goods procurement status
       - Contract awards
   4.3 Disbursement
       - Financial summary table
       - Disbursement rate analysis

5. Safeguards and Cross-cutting
   - Environmental safeguards
   - Social safeguards
   - Gender action plan progress

6. Risk Assessment
   - Updated risk matrix
   - New risks identified
   - Mitigation measures status

7. Next Quarter Work Plan
   - Planned activities
   - Upcoming milestones
   - Expert deployment schedule

Annexes
   - Updated work plan (Gantt chart)
   - Financial statements
   - Photo documentation
```

## ADB 보고 시 유의사항

- 보고 주기: 반기별 (Semi-annual) 기본, 일부 TA는 분기별
- 제출처: ADB Project Officer + Executing Agency
- 언어: 영문
- DMF 기반: Design and Monitoring Framework 지표 중심 보고
- Disbursement: 재정 집행률을 ADB가 매우 중시 — 집행률이 낮으면 프로젝트 지연 경고
- Traffic Light System: Green/Amber/Red로 각 항목 상태 표시
