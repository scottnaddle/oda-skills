# KOICA 분기 진도보고서 양식

## 표준 목차

```
[사업명] 분기 진도보고서
보고 기간: YYYY.Q# (MM~MM월)

1. 사업 개요
   - 사업명/기간/예산/수행기관

2. 사업 추진 실적 요약
   - 전체 진도율
   - 주요 성과 (3~5건)
   - 주요 이슈 (1~3건)

3. 산출물별 추진 실적
   3.1 산출물 1: [명칭]
       - 계획 vs 실적
       - 상세 활동 내역
       - 성과지표 달성 현황
   3.2 산출물 2: [명칭]
       ...

4. 범분야 이슈 추진 실적
   - 젠더: 여성 참여 현황
   - 환경: 환경 고려 실적
   - 기타

5. 예산 집행 현황
   - 비목별 집행 현황표
   - 집행률 분석
   - 비목 전용 내역 (해당 시)

6. 전문가 투입 현황
   - 전문가별 투입 실적 (M/M)
   - 파견 일정 및 활동 요약

7. 이슈 및 위험관리
   - 현안 이슈 및 대응
   - 리스크 등록부 업데이트
   - 발주처 협의 필요 사항

8. 차기 분기 계획
   - 핵심 활동 일정
   - 성과물 제출 예정
   - 전문가 파견 계획

9. 첨부
   - 활동 사진
   - 참석자 명단
   - 산출물 샘플
```

## KOICA 보고 시 유의사항

- 보고 주기: 분기별 (일부 사업은 월별)
- 제출 기한: 보고 기간 종료 후 15~30일 이내
- 제출처: KOICA 담당 부서 + 현지 사무소
- 언어: 국문 (일부 사업 국/영 병행)
- 성과지표: PDM 기반으로 정량적 보고
- PMC 사업: PMC가 시행기관의 보고를 취합하여 KOICA에 보고
