# 수원국 니즈 평가

수원국의 교육 분야 니즈를 체계적으로 분석하여 사업 기획의 근거를 마련한다.

## 분석 프레임워크

### 1. 현황-목표 갭 분석
- 현재 상태(As-Is): 교육 통계, 정책 현황, 인프라 수준
- 목표 상태(To-Be): 국가 교육 전략 목표, SDG4 목표, 국제 벤치마크
- 갭(Gap): 현황과 목표 사이의 차이가 곧 "니즈"

### 2. 니즈 우선순위화
갭을 식별한 후, 다음 기준으로 우선순위를 매긴다:
- **시급성**: 즉시 해결이 필요한가?
- **규모**: 영향받는 인구/기관의 수
- **실현가능성**: 외부 개입으로 해결 가능한가?
- **정책 합치성**: 수원국 정부의 우선순위와 맞는가?
- **UBION 적합성**: UBION이 기여할 수 있는 영역인가?

### 3. 수요-공급 매핑
- 수요(Demand): 수원국이 필요로 하는 것
- 공급(Supply): 이미 제공되고 있는 ODA 사업
- 공백(Unmet Need): 아직 충족되지 않은 니즈 → 사업 기회

## 출력 형식

```
# 니즈 평가: [국가명] [분야]

## 1. 갭 분석 매트릭스
## 2. 니즈 우선순위 표
## 3. 수요-공급 매핑
## 4. UBION 사업 기회 Top 3
```
