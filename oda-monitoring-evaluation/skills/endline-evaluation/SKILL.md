# 종료평가 설계 및 보고

사업 종료 시점에 실시하는 총괄적(Summative) 평가를 설계하고 보고한다.

## 종료평가의 목적

중간평가가 "방향 교정"이라면, 종료평가는 "성적표"다. 사업이 약속한 성과를 실제로 달성했는지, 투입한 자원이 효과적으로 사용되었는지, 효과가 지속될 것인지를 종합 판단한다. 결과는 발주처의 사업 관리 개선과 후속 사업 기획에 활용된다.

## DAC 6대 기준 평가 프레임워크

### 적절성 (Relevance)
- 사업 목표가 수원국 니즈와 부합하였는가?
- 수원국 정책/전략과 정합성이 있었는가?
- 사업 기간 중 니즈 변화에 적응했는가?
- 수혜자 선정이 적절했는가?

### 일관성 (Coherence)
- 내적 일관성: 사업 내 구성요소 간 시너지가 있었는가?
- 외적 일관성: 다른 ODA 사업, 수원국 자체 사업과 조화되었는가?
- SDGs/발주처 전략과의 정합성

### 효과성 (Effectiveness)
- 계획된 산출물이 달성되었는가? (PDM 지표 기반)
- 의도한 성과(Outcome)가 달성되었는가?
- 미달성 항목의 원인은 무엇인가?
- 의도하지 않은 결과(긍정/부정)가 있었는가?

### 효율성 (Efficiency)
- 예산이 계획대로 집행되었는가?
- 투입 대비 산출이 적정했는가?
- 일정이 계획대로 진행되었는가?
- 더 효율적인 대안이 있었는가?

### 영향력 (Impact)
- 상위 목표(Impact Level)에 어떤 기여를 했는가?
- 사업이 만든 변화의 범위와 깊이는?
- 장기적 파급효과 전망
- 사업이 없었다면 어떤 차이가 있었을 것인가? (Counterfactual)

### 지속가능성 (Sustainability)
- 사업 효과가 종료 후에도 지속될 가능성은?
- 제도적 지속가능성: 법/제도/조직 차원 정착 여부
- 재정적 지속가능성: 수원국 자체 예산 확보 여부
- 기술적 지속가능성: 현지 인력의 운영/유지보수 역량
- 사회적 지속가능성: 수혜자의 지속적 참여 의지

## 종료평가 보고서 구조

```
# 종료평가 보고서: [사업명]

## Executive Summary (3~5쪽)

## 1. 평가 개요
1.1 평가 목적 및 범위
1.2 평가 방법론
1.3 평가팀
1.4 제약사항

## 2. 사업 개요
2.1 사업 배경
2.2 추진 경과
2.3 주요 변경 사항

## 3. DAC 기준별 평가
3.1 적절성
3.2 일관성
3.3 효과성 (PDM 성과 달성 분석 포함)
3.4 효율성
3.5 영향력
3.6 지속가능성

## 4. 범분야 이슈 (KOICA 사업)
4.1 젠더 평등
4.2 환경
4.3 인권/장애포용

## 5. 종합 결론

## 6. 교훈

## 7. 제언
[대상별, 우선순위별 구조화]

## 별첨
```

## 평가 등급 체계

각 DAC 기준에 대해 4단계 또는 6단계 등급을 부여한다:

**4단계**
| 등급 | 기준 |
|---|---|
| 매우 우수 | 기대를 크게 초과 달성 |
| 우수 | 대부분의 목표 달성 |
| 보통 | 일부 목표 달성, 개선 필요 |
| 미흡 | 대부분의 목표 미달성 |

## 다음 단계 제안

- **영향평가 방법론이 필요하면** → impact-assessment 스킬
- **설문 데이터를 분석하려면** → survey-analysis 스킬
- **종료보고서와 통합하려면** → completion-report 스킬 (oda-project-execution 플러그인)
