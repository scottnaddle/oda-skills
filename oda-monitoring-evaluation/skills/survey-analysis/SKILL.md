# 설문 데이터 분석

ODA 사업의 설문조사 데이터를 체계적으로 분석하여 의미 있는 발견사항과 시사점을 도출한다.

## 분석 절차

### 1단계: 데이터 확인 및 정제

- 응답 건수 확인 (유효/무효 구분)
- 결측값(Missing Data) 현황 파악
- 이상값(Outlier) 확인
- 변수 유형 확인 (범주형/연속형)
- 성별, 지역 등 분류변수 확인

### 2단계: 기술통계 (Descriptive Statistics)

각 문항에 대해:
- **범주형 변수**: 빈도, 비율, 최빈값
- **연속형 변수**: 평균, 중앙값, 표준편차, 최소/최대
- **리커트 척도**: 평균, 분포 (매우 동의~매우 비동의), Top2/Bottom2 비율

### 3단계: 교차분석 (Cross-tabulation)

주요 분류변수별 차이 분석:
- 성별(남/여)에 따른 차이
- 지역(도시/농촌)에 따른 차이
- 경력/직급에 따른 차이
- 사전/사후 시점에 따른 차이

통계적 유의성 검정:
- 범주형×범주형: 카이제곱 검정
- 범주형×연속형: t-검정, ANOVA
- 사전-사후: 대응표본 t-검정, Wilcoxon 부호순위 검정

### 4단계: 만족도 분석 (해당 시)

연수/워크숍/서비스 만족도 분석:
- 전체 만족도 점수 및 분포
- 항목별 만족도 비교 (내용, 강사, 시설, 자료 등)
- 만족/불만족 사유 분석 (자유응답 연계)
- IPA(Importance-Performance Analysis) — 중요도와 만족도 동시 분석

### 5단계: 사전-사후 비교 (해당 시)

연수/교육 효과 측정:
- 사전/사후 테스트 점수 비교
- 효과 크기(Effect Size) 산출 — Cohen's d
- 항목별 변화량 분석
- 하위 집단별 효과 차이

### 6단계: 자유응답 분석

개방형 질문 응답 분석:
- 주제(Theme) 분류 및 코딩
- 빈도 기반 핵심 주제 도출
- 대표 인용문 선별
- 워드클라우드 (시각화)

### 7단계: 시각화 및 보고서 작성

분석 결과를 시각적으로 정리:
- 막대/원형 그래프: 빈도 분포
- 평균 비교 차트: 항목별, 집단별 비교
- 사전-사후 변화 그래프: 꺾은선/막대
- 히트맵: 교차분석 결과

## 출력 형식

```
# 설문 분석 보고서: [조사명]

## 1. 조사 개요
- 조사 기간, 대상, 방법, 응답 현황

## 2. 응답자 프로파일
[인구통계학적 특성 요약]

## 3. 주요 발견사항
### 3.1 [영역 1]
[분석 결과 + 차트]

### 3.2 [영역 2]
[분석 결과 + 차트]

## 4. 교차분석 결과
[집단 간 차이 분석]

## 5. 자유응답 분석
[핵심 주제 + 대표 인용문]

## 6. 시사점 및 제언
[분석에서 도출된 액션 아이템]

## 별첨: 상세 통계표
```
