# PCM 성과지표 상세 가이드

## 지표 유형

### 정량 지표 (Quantitative)
수치로 측정 가능한 지표. ODA에서 선호되는 유형.

**예시:**
- 연수 이수자 수 (명)
- ICT 활용 수업 비율 (%)
- LMS 접속 건수 (건/월)
- 시험 점수 평균 (점)
- 취업률 (%)

### 정성 지표 (Qualitative)
수치화가 어렵지만 질적 변화를 포착하는 지표.

**예시:**
- 교원의 ICT 활용 수업 역량 수준 (상/중/하)
- 교육과정의 산업 현장 적합성 (평가 등급)
- 이해관계자 만족도 (리커트 5점 척도)
- 정책 채택 여부 (채택/미채택)

### 대리 지표 (Proxy)
직접 측정이 어려울 때 대리하는 지표.

**예시:**
- "교육 품질 향상"의 대리 → 학생 학업성취도, 교원 역량 점수
- "디지털 전환 수준"의 대리 → LMS 활성 사용자 비율, 온라인 콘텐츠 이용률

## SMART 원칙 상세

### Specific (구체적)
나쁜 예: "교육 역량 향상"
좋은 예: "대상 학교 교원의 ICT 활용 수업 실시 역량"

무엇을(What), 누구를(Who), 어디서(Where) 측정하는지 명확해야 한다.

### Measurable (측정 가능)
나쁜 예: "수업의 질 개선"
좋은 예: "수업관찰 평가 점수 평균 70점 이상 달성 (100점 만점)"

측정 도구, 척도, 단위가 정의되어야 한다.

### Achievable (달성 가능)
나쁜 예: "2년 내 전국 교원 100% ICT 역량 확보"
좋은 예: "대상 지역 교원 200명 중 80% 이상 연수 이수"

사업 기간, 예산, 역량 범위 내에서 현실적이어야 한다.

### Relevant (적절)
나쁜 예: (디지털 교육 사업에서) "교실 건축 수"
좋은 예: (디지털 교육 사업에서) "디지털 교과서 활용 수업 비율"

사업 목표와 직접적으로 연결되어야 한다.

### Time-bound (시한)
나쁜 예: "교원 역량 향상"
좋은 예: "사업 종료 시(2028년 12월) 교원 역량 평가 점수 30% 향상"

달성 시점이 명시되어야 한다.

## 교육 분야 표준 지표 라이브러리

### Output 수준 지표

| 영역 | 지표 예시 | 단위 |
|---|---|---|
| 연수/워크숍 | 연수 이수자 수 (성별 분리) | 명 |
| | 연수 만족도 | 5점 척도 |
| | 연수 사전-사후 역량 변화 | 점수 차 |
| 콘텐츠 개발 | 개발된 교육 콘텐츠 모듈 수 | 건 |
| | 콘텐츠 품질 검증 통과율 | % |
| 시스템 구축 | LMS 구축 완료 | 식 |
| | 시스템 가용률(Uptime) | % |
| | 등록 사용자 수 | 명 |
| 정책/제도 | 개발된 가이드라인/매뉴얼 수 | 건 |
| | 정부 승인/채택 여부 | Y/N |

### Outcome 수준 지표

| 영역 | 지표 예시 | 단위 |
|---|---|---|
| 역량 변화 | ICT 활용 수업 실시 교원 비율 | % |
| | 교원 자기효능감 점수 변화 | 점수 차 |
| 행동 변화 | LMS 월간 활성 사용자 비율 | % |
| | 디지털 교수법 적용 수업 비율 | % |
| 제도 변화 | ICT 교육 정책/규정 제정 건수 | 건 |
| | ICT 교육 예산 배정 변화 | % |

### Impact 수준 지표

| 영역 | 지표 예시 | 단위 |
|---|---|---|
| 교육 성과 | 학생 학업성취도 변화 | 점수 차 |
| | 취학률/중퇴율 변화 | %p |
| 고용 성과 | 졸업생 취업률 변화 | %p |
| | 취업 후 6개월 유지율 | % |
| 시스템 성과 | 교육 품질 지수 변화 | 지수 |
| | 디지털 교육 보급률 | % |

## 성별 분리 데이터 (Gender-Disaggregated Data)

KOICA와 대부분의 공여기관은 성별 분리 데이터를 요구한다:
- 모든 인원 관련 지표는 남/여 분리 수집
- 성별 격차(Gender Gap) 변화를 추적하는 별도 지표 고려
- "여성 참여율 XX% 이상" 같은 젠더 특화 지표 설정

## 데이터 수집 도구

| 도구 | 적합한 지표 유형 | 장점 | 한계 |
|---|---|---|---|
| 설문조사 | 만족도, 인식, 행동 변화 | 대규모, 정량화 용이 | 응답 편향 |
| 사전-사후 테스트 | 역량/지식 변화 | 변화 측정 명확 | 테스트 효과 |
| 수업 관찰 | 교수법 적용, 교실 환경 | 실제 행동 관찰 | 관찰자 편향, 비용 |
| 인터뷰/FGD | 심층 인식, 맥락 이해 | 풍부한 정성 데이터 | 시간/비용, 일반화 어려움 |
| 시스템 로그 | LMS 접속, 콘텐츠 이용 | 객관적, 실시간 | 활용 품질 미측정 |
| 행정 데이터 | 취학률, 졸업률, 취업률 | 공식, 비용 저렴 | 지연, 정확도 이슈 |
