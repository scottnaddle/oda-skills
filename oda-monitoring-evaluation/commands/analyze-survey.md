# /analyze-survey

설문조사 데이터의 품질 점검부터 분석, 보고서 작성까지 한 번에 실행하는 워크플로우.

## 실행 흐름

### Step 1: 데이터 품질 점검 (data-quality-check)

분석 전에 데이터의 신뢰성을 확인한다.
- 응답 건수 및 결측값 현황
- 이상값 식별
- 활용 가능 판단

### Step 2: 설문 분석 (survey-analysis)

체계적 분석을 실시한다.
- 기술통계 (빈도, 평균, 분포)
- 교차분석 (성별, 지역, 집단별 차이)
- 만족도 분석 (해당 시)
- 사전-사후 비교 (해당 시)
- 자유응답 분석
- 시각화

## 완료 후 산출물

설문 분석 보고서 (조사개요, 프로파일, 발견사항, 시사점)

## 완료 후 제안

- 중간/종료평가에 통합하려면: → `/run-midterm` 또는 `/run-endline`
- 진도보고에 반영하려면: → `/write-progress` (oda-project-execution 플러그인)
