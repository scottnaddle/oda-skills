# /draft-email

상황과 수신자에 맞는 업무 이메일을 작성하는 워크플로우.

## 실행 흐름

email-draft 스킬을 실행한다.

1. 수신자 유형 확인 (발주처/카운터파트/전문가/파트너)
2. 용건 파악
3. 적절한 톤과 구조 선택
4. 초안 작성 (한글 또는 영문)
5. 제목(Subject line) 포함

## 완료 후 제안

- 번역이 필요하면: → `/translate`
- 문법 교정이 필요하면: → `/proofread`
