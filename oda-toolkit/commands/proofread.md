# /proofread

ODA 문서의 문법과 스타일을 교정하는 워크플로우.

## 실행 흐름

grammar-check-ko-en 스킬을 실행한다.

1. 문서 언어 식별 (한글/영문/혼용)
2. 1차: 문법/맞춤법 오류 식별
3. 2차: 문체/스타일 개선
4. 3차: ODA 특화 표현 점검 (추상적 서술, 번역투 등)
5. 4차: 일관성 및 형식 점검
6. 변경 사항 표 + 교정본 제시

## 완료 후 제안

- 번역이 필요하면: → `/translate`
- 제안서 전체를 검토하려면: → `/review-proposal` (oda-proposal-writing 플러그인)
