# 계약서 검토

ODA 사업 수행 관련 계약서의 핵심 조항을 검토하고, 주의가 필요한 리스크 포인트를 식별한다.

이 스킬은 법률 자문을 대체하지 않는다. 핵심 체크포인트를 빠르게 정리하여, 법률 검토 전 사전 확인을 지원하는 도구다.

## 검토 체크리스트

### 1. 계약 기본 사항
- [ ] 계약 당사자 (정확한 법인명, 대표자)
- [ ] 계약 목적 및 범위
- [ ] 계약 기간 (시작일, 종료일, 연장 조건)
- [ ] 계약 금액 및 통화

### 2. 업무 범위 (Scope of Work)
- [ ] 업무 범위가 명확하고 구체적인가
- [ ] 제안서/TOR과 일치하는가
- [ ] 범위 변경(Variation) 절차가 명시되어 있는가
- [ ] 제외 사항이 명확한가

### 3. 지불 조건
- [ ] 지불 일정 (선급, 중도, 잔금 비율)
- [ ] 지불 조건 (성과물 제출 후 X일 이내)
- [ ] 지체 지급 시 이자/페널티
- [ ] 환율 기준 (고정/변동)
- [ ] 세금 처리 (면세 여부)

### 4. 성과물 및 검수
- [ ] 성과물 목록 및 제출 일정
- [ ] 검수 기준 및 절차
- [ ] 검수 기간
- [ ] 보완 요청 절차

### 5. 지적재산권
- [ ] 성과물 저작권 귀속 (발주처/수행기관/공동)
- [ ] 기존 IP(배경지식) 보호
- [ ] 제3자 SW 라이선스 처리
- [ ] 공개/비공개 조건

### 6. 리스크 조항
- [ ] 불가항력(Force Majeure) 정의 및 면책
- [ ] 지체상금 조항 (비율, 상한)
- [ ] 계약 해지 조건 (쌍방/일방)
- [ ] 하자보수 기간 및 범위
- [ ] 보증(Warranty) 조건
- [ ] 손해배상 한도

### 7. 분쟁 해결
- [ ] 준거법 (어느 나라 법)
- [ ] 분쟁 해결 방식 (협의→중재→소송)
- [ ] 중재 기관 및 장소

### 8. 기타
- [ ] 비밀유지 조항
- [ ] 하도급 조건
- [ ] 보험 요구사항
- [ ] 반부패 조항
- [ ] 양도 제한

## 출력 형식

```
# 계약서 검토 보고서: [계약명]

## 1. 계약 개요
[기본 정보 요약]

## 2. 체크리스트 결과
[항목별 ✅/⚠️/❌]

## 3. 리스크 포인트
| # | 조항 | 리스크 | 심각도 | 권고 |
|---|---|---|---|---|
| 1 | [조항] | [리스크] | 높음/중간/낮음 | [조치] |

## 4. 수정 협상 포인트
[발주처와 협의가 필요한 사항]

## 5. 법률 검토 필요 사항
[전문 법률 자문이 필요한 항목]
```
