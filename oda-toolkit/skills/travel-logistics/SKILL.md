# 출장 일정 관리

ODA 사업 현지 출장의 전 과정(준비→이동→현지→귀국)을 체계적으로 관리한다.

## 출장 계획 구조

### 1. 출장 개요
- 출장 목적 및 기대 성과
- 출장지 (국가, 도시)
- 출장 기간
- 출장자 명단

### 2. 사전 준비 체크리스트

**서류/행정**
- [ ] 여권 유효기간 확인 (6개월 이상)
- [ ] 비자 발급/면제 확인
- [ ] 출장 품의서/결재
- [ ] 해외여행보험 가입
- [ ] 예방접종 확인 (말라리아, 황열병 등 — 국가별)
- [ ] 출장비 가불/카드 준비

**항공/숙소**
- [ ] 항공권 예약 (왕복, 경유지 확인)
- [ ] 숙소 예약 (미팅 장소 접근성 고려)
- [ ] 공항 픽업/샌딩 수배
- [ ] 현지 교통 수배 (렌터카/기사)

**업무 준비**
- [ ] 미팅 일정 확정 및 상대방 확인
- [ ] 발표 자료/배포 자료 준비
- [ ] 통역 수배 (필요 시)
- [ ] 선물/기념품 (프로토콜에 따라)
- [ ] 명함 (영문/현지어)

### 3. 현지 일정표

```

|---|---|---|---|---|---|
| D-1 | 22:00 | 인천 출발 | ICN | | OZ XXX |
| D+0 | 06:00 | 현지 도착 | 호텔 체크인 | 픽업 | |
| D+0 | 10:00 | 교육부 면담 | 교육부 청사 | 차량 | 통역 동행 |
| D+0 | 14:00 | 현장 방문 | XX학교 | 차량 | 사진 촬영 |
| ... | ... | ... | ... | ... | ... |
```

### 4. 국가별 유의사항

출장지별로 확인해야 할 사항:
- 현지 공휴일/종교 행사 (라마단, 나우르즈 등)
- 기후/복장 (우기, 극한 기온)
- 안전 정보 (외교부 여행경보 단계)
- 전압/플러그 규격
- 현지 통화/환전
- 긴급 연락처 (대사관, 현지 사무소)
- 문화적 에티켓 (인사법, 선물, 식사)
