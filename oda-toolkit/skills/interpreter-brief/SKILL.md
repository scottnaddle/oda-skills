# 통역 브리핑 자료 작성

ODA 사업 회의, 워크숍, 세미나 등에서 통역사가 사전에 숙지해야 할 브리핑 자료를 작성한다.

## 브리핑 자료 구조

### 1. 행사 개요
- 행사명, 일시, 장소
- 참석자 (한국 측 / 현지 측)
- 통역 방식 (순차/동시), 통역 언어
- 전체 일정 및 통역 세션

### 2. 핵심 용어 대조표

```
| 한국어 | 영어 | 현지어(해당 시) | 비고 |
|---|---|---|---|
| 역량강화 | Capacity Building | | 자주 사용 |
| 로지컬 프레임워크 | Logical Framework | | 약어: Logframe |
| 운영위원회 | PSC (Project Steering Committee) | | |
| 산출물 | Output | | Outcome과 구분 필요 |
| [사업 특수 용어] | [영문] | [현지어] | [설명] |
```

### 3. 약어 목록

```
| 약어 | 풀네임 | 한국어 |
|---|---|---|
| KOICA | Korea International Cooperation Agency | 한국국제협력단 |
| PMC | Project Management Consulting | 사업관리컨설팅 |
| TOR | Terms of Reference | 과업지시서 |
| PDM | Project Design Matrix | 사업설계매트릭스 |
| M&E | Monitoring and Evaluation | 모니터링·평가 |
| [사업별 약어] | | |
```

### 4. 주요 발언 예상 내용
- 한국 측 주요 발언 포인트 (미리 공유 가능한 범위)
- 현지 측 예상 관심사/질문
- 발표 자료 사전 공유 (가능한 경우)

### 5. 민감 사안 가이드
- 정치적으로 민감한 표현 (피해야 할 용어)
- 문화적 주의사항
- 숫자/통계 인용 시 정확성 주의
- 합의/약속과 단순 의견 교환의 구분

### 6. 실용 정보
- 통역 장비 (마이크, 이어폰) 확인
- 자료 배치 (통역석 앞 용어집, 발표자료)
- 휴식 시간 (동시통역 시 교대 일정)
- 비상 연락처
