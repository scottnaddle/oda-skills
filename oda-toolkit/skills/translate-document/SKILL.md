# ODA 문서 번역

ODA 사업 관련 문서를 전문성과 일관성을 유지하며 번역한다.

## 번역 원칙

### 1. 의미 전달 우선
직역보다 의미가 정확히 전달되는 번역을 우선한다. 단, 고유명사, 기관명, 프로그램명은 공식 번역을 사용한다.

### 2. 전문 용어 일관성
동일 문서 내에서 같은 용어는 반드시 같은 번역어를 사용한다.

**ODA 핵심 용어 대조표:**

| 한국어 | 영어 |
|---|---|
| 수원국 | Partner Country / Recipient Country |
| 공여기관 | Donor Agency |
| 로지컬 프레임워크 | Logical Framework (Logframe) |
| 성과지표 | Performance Indicator |
| 기초선 | Baseline |
| 목표치 | Target |
| 산출물 | Output |
| 성과 | Outcome |
| 영향 | Impact |
| 역량강화 | Capacity Building / Capacity Development |
| 기술협력 | Technical Cooperation |
| 사업관리컨설팅 | Project Management Consulting (PMC) |
| 과업지시서 | Terms of Reference (TOR) |
| 운영위원회 | Project Steering Committee (PSC) |
| 카운터파트 | Counterpart |
| 범분야 이슈 | Cross-cutting Issues |
| 젠더 평등 | Gender Equality |
| 지속가능성 | Sustainability |
| 출구전략 | Exit Strategy |
| 중간평가 | Mid-term Evaluation |
| 종료평가 | End-of-Project Evaluation |

### 3. 발주처별 선호 표현
- KOICA: "수원국", "성과중심관리", "사업목적"
- ADB: "Developing Member Country (DMC)", "Results-Based Management", "Project Purpose"
- UNESCO: "Member State", "Programme", "Education Sector"

### 4. 문체
- 한→영: 수동태 적절 활용, 공식 문체, 간결한 문장
- 영→한: 능동태 선호, 존대체(합쇼체), 보고서 격식체

## 번역 유형별 가이드

### 제안서 번역
- 설득력 있는 톤 유지
- 기술 용어의 정확성 최우선
- 배점 항목 키워드가 번역에서 누락되지 않도록 주의

### 보고서 번역
- 객관적 톤 유지
- 수치/통계의 정확한 전환 (통화 단위 포함)
- 성과 관련 표현의 일관성

### 공식 서한/이메일 번역
- 외교적 관례 준수
- 수신/발신자 직위 정확히 표기
- 첨부 서류 언급 누락 주의
