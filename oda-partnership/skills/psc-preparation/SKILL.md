# PSC 회의 준비

PSC(운영위원회) 회의를 체계적으로 준비하여, 효과적인 의사결정과 사업 방향 합의를 이끌어낸다.

## PSC란

PSC는 ODA 사업의 최고 의사결정 기구다. 발주처, 수원국 카운터파트, 수행기관의 고위급이 참석하여 사업 방향, 주요 이슈, 계획 변경을 승인한다. 보통 반기 또는 연 1~2회 개최.

## 준비 자료 패키지

### 1. 회의 안건 (Agenda)

```
PSC 제[X]차 회의 안건

일시: YYYY.MM.DD HH:MM
장소: [장소]
참석자: [발주처/카운터파트/수행기관 고위급]

1. 개회 (인사말)
2. 전차 회의 후속조치 이행 현황
3. 사업 추진 현황 보고
4. 주요 이슈 및 의사결정 사항
   4.1 [안건 1: 승인/결정 필요 사항]
   4.2 [안건 2: 협의 사항]
5. 차기 계획
6. 기타 사항
7. 폐회
```

### 2. 사업 추진 현황 요약
- 전체 진도율 (간결하게)
- 산출물별 달성 현황 (표 1장)
- 예산 집행 현황 (표 1장)
- 주요 성과 하이라이트 (3~5건, 사진 포함)
- progress-report 스킬의 산출물을 요약

### 3. 의사결정 필요 사항

각 안건에 대해:
```
안건: [제목]
배경: [왜 이 안건이 필요한가 — 2~3줄]
선택지:
  A안: [내용] — 장점: / 단점:
  B안: [내용] — 장점: / 단점:
수행기관 권고: [A안 또는 B안, 근거]
요청: [승인/결정/참고 — 어떤 수준의 의사결정이 필요한지]
```

### 4. 전차 회의 후속조치 이행 현황
```
| # | 후속조치 | 담당 | 기한 | 이행 현황 | 비고 |
|---|---|---|---|---|---|
| 1 | [조치] | [담당] | [기한] | 완료/진행중/지연 | |
```

### 5. 발표 자료 (PPT)
- 10~15슬라이드 이내
- 고위급 대상이므로 핵심만 간결하게
- 시각적 요소 활용 (그래프, 사진, 인포그래픽)
- presentation-script 스킬 연계 가능

## 출력 형식

```
# PSC 제[X]차 회의 준비 자료: [사업명]

## 1. 안건 (Agenda)
## 2. 추진 현황 요약
## 3. 의사결정 사항
## 4. 전차 후속조치 이행
## 5. 발표 자료 구성안
## 6. 로지스틱스 (장소, 다과, 통역, 장비)
```
