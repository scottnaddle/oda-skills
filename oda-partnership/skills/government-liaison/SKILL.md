# 수원국 정부 대응 전략

ODA 사업에서 수원국 정부 기관(카운터파트)과의 효과적인 소통과 협력 관계를 관리한다.

## 정부 대응 프레임워크

### 1. 카운터파트 기관 이해
- 조직 구조 (부서, 직급 체계)
- 의사결정 프로세스 (누가, 어떤 절차로 결정하는가)
- 업무 문화 (회의 관행, 소통 방식, 속도)
- 핵심 인물 프로파일 (stakeholder-mapping 연계)

### 2. 소통 전략
- **공식 채널**: 공문(Official Letter), 공식 미팅
- **실무 채널**: 이메일, 전화, 메신저 (현지 관행에 따라)
- **비공식 채널**: 식사, 행사, 소규모 모임
- **빈도**: 고위급 분기 1회, 실무급 주 1회 이상

### 3. 공식 서한 작성

ODA 사업에서 자주 필요한 공식 서한 유형:
- 사업 착수 통보/협력 요청
- 전문가 파견 통보
- 자료/정보 요청
- 성과물 제출 및 검수 요청
- 감사 서한
- 초청장 (연수, 워크숍)

**공식 서한 기본 구조:**
```
[발신 기관 로고/레터헤드]
Date: [날짜]
Ref: [참조번호]

To: [수신자 직위, 성명]
    [기관명]

Subject: [건명]

Dear [Title] [Name],

[인사/서두]

[본문 — 목적, 배경, 요청사항]

[마무리 — 감사, 기대, 연락처]

Sincerely,

[서명]
[발신자 성명]
[직위]
[기관명]
[연락처]

CC: [참조 대상]
Encl: [첨부 목록]
```

### 4. 정부 승인 프로세스 관리

ODA 사업에서 정부 승인이 필요한 항목:
- 사업 워크플랜 변경
- 주요 인력 변경
- 예산 비목 전용
- 성과물(커리큘럼, 가이드라인) 공식 채택
- 기자재 소유권 이전
- 사업 연장

각 승인 항목에 대해:
- 승인 권한자 확인
- 필요 서류 목록
- 예상 소요 기간
- 촉진(Facilitation) 방법

### 5. 관계 리스크 관리

- **인사 이동**: 카운터파트 핵심 인물이 교체될 때 → 다층적 관계 구축
- **정치 변화**: 정권/장관 교체 시 정책 변화 가능 → 실무선 연속성 확보
- **소통 단절**: 응답이 없을 때 → 현지 사무소/현지 직원 활용
- **이해 충돌**: 사업 방향에 대한 의견 차이 → PSC에서 공식 합의

## 출력 형식

```
# 정부 대응 전략: [사업명] - [카운터파트 기관명]

## 1. 기관 이해
## 2. 소통 계획
## 3. 공식 서한 (해당 시)
## 4. 승인 필요 사항 추적표
## 5. 리스크 및 대응
```
