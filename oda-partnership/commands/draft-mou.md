# /draft-mou

파트너십 MOU/MOA 초안을 작성하는 워크플로우.

## 실행 흐름

mou-draft 스킬을 실행한다.

1. MOU vs MOA 유형 결정
2. 당사자 정보 확인
3. 협력 범위 및 역할 정의
4. 표준 조항 적용 (기간, IP, 비밀유지, 해지, 분쟁해결)
5. 한영 이중언어 초안 생성

## 완료 후 제안

- 법무 검토를 권고한다
- 하도급 범위를 정하려면: → subcontract-scope 스킬
